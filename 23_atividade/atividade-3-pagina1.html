Design Digital: um guia para estudantes pré-universitários

Informações gerais

Turno: Diurno

Titulação: Bacharelado

Duração: 4 anos (3008h/aula)

Objetivos do curso

Formar profissionais aptos a criar, organizar, planejar e produzir (inclusive programando) interfaces de interação com usuário para produtos de software.

Áreas de atuação

Atua como membro de equipes de desenvolvimento de software, sendo responsável pela concepção visual e funcional das interfaces dos sistemas digitais.
